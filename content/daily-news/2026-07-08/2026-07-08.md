## 1. Służby ruszają ze szkoleniami z AI, dezinformacji i cyberhigieny

- **Kategoria:** Polska
- **Podsumowanie:** Ministerstwo Cyfryzacji uruchomiło program szkoleń z zaawansowanych kompetencji cyfrowych dla funkcjonariuszy i pracowników Policji, Państwowej Straży Pożarnej oraz Straży Granicznej. Zakres obejmuje wykorzystanie AI, przeciwdziałanie dezinformacji, cyberbezpieczeństwo, cyberhigienę i identyfikowanie technik manipulacyjnych. W 2026 r. zaplanowano przeszkolenie minimum 1750 osób oraz przekazanie co najmniej 982 zestawów urządzeń do uwierzytelniania dwuskładnikowego.
- **Data publikacji:** 7 lipca 2026

## 2. Ministerstwo Cyfryzacji zapowiada spotkanie o Gigafabryce AI

- **Kategoria:** Polska

## 3. Kimi K2.7 trafia do GitHub Copilot Business i Enterprise

- **Kategoria:** Świat
- **Podsumowanie:** GitHub rozszerzył dostępność modelu Kimi K2.7 Code na plany Copilot Business i Copilot Enterprise. Model jest prezentowany jako pierwsza otwartowagowa opcja w selektorze modeli GitHub Copilot, hostowana przez GitHub na Microsoft Azure. W organizacjach model jest domyślnie wyłączony i wymaga włączenia przez administratora po ocenie wymagań bezpieczeństwa, zgodności i zarządzania danymi.
- **Data publikacji:** 7 lipca 2026
- **Data zdarzenia:** 7 lipca 2026 — rozszerzenie dostępności modelu dla planów biznesowych i enterprise.
- **Źródło:** GitHub Changelog
- **URL:** https://github.blog/changelog/2026-07-07-kimi-k2-7-now-available-for-copilot-business-and-enterprise/

## 4. Rząd Alberty pokazuje skalę użycia Claude Code w cyberbezpieczeństwie

- **Kategoria:** Świat
- **Podsumowanie:** Anthropic opisał przypadek administracji Alberty, która użyła Claude Code z modelami Opus i Sonnet do przeglądu systemów publicznych. Zespół przeskanował 466 mln linii kodu w 20 godzin, wskazując podatności, braki dokumentacyjne i problemy modernizacyjne. Wdrożenie pokazuje, jak agenci AI mogą działać w administracji jako narzędzia do audytu repozytoriów, generowania poprawek, pisania testów i planowania modernizacji systemów.
- **Data publikacji:** 6 lipca 2026
- **Data zdarzenia:** od 2025 r.; publikacja case study 6 lipca 2026
- **Źródło:** Anthropic
- **URL:** https://www.anthropic.com/news/alberta-government-claude-cybersecurity

## 5. GitHub Copilot w VS Code dostaje ogólnodostępne narzędzia przeglądarki

- **Kategoria:** Świat
- **Podsumowanie:** GitHub ogłosił ogólną dostępność narzędzi przeglądarkowych dla Copilota w VS Code. Agenci mogą otwierać strony, klikać, wpisywać dane, obsługiwać okna dialogowe, czytać treść stron, wykonywać zrzuty ekranu i analizować błędy konsoli. Jednocześnie prywatne karty użytkownika nie są automatycznie dostępne dla agenta, a karty agentów działają w izolowanych sesjach.
- **Data publikacji:** 1 lipca 2026; nota redakcyjna z doprecyzowaniem 1 lipca 2026
- **Data zdarzenia:** 1 lipca 2026 — ogólna dostępność funkcji.
- **Źródło:** GitHub Changelog
- **URL:** https://github.blog/changelog/2026-07-01-browser-tools-for-github-copilot-in-vs-code-are-generally-available/
