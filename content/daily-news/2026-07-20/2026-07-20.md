- **Kategoria:** Polska

## 2. Microsoft ogłasza pełną integrację autonomicznych subagentów AI w środowisku GitHub Copilot i VS Code

- **Kategoria:** Świat
- **Podsumowanie:** Microsoft ogłosił przełomową aktualizację środowiska VS Code oraz usługi GitHub Copilot. Wprowadzono wsparcie dla tła wykonywania zadań przez autonomiczne subagenty, które potrafią w tle uruchamiać testy jednostkowe, naprawiać linterowe błędy i generować kompletne Pull Requesty w odizolowanych gałęziach środowiskowych.
